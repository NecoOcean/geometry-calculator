# 几何计算器

这是一个基于 Vue.js 的几何计算器，可以计算圆锥和圆台的表面积、体积、重量和成本。

## 功能特点

- 支持圆锥和圆台的计算
- 可配置材料参数（密度、价格）
- 支持多种单位转换
- 显示详细的计算步骤
- 美观的用户界面

## 使用技术

- Vue.js 3
- Composition API
- Tailwind CSS

## 本地开发

1. 克隆项目
```bash
git clone [repository-url]
```

2. 安装依赖
```bash
npm install
```

3. 启动开发服务器
```bash
npm run dev
```
